# FlightOps Academy

**Aviation Operations Knowledge Base**
A professional study and reference guide for Airport Operations Officers, Aerodrome Reporting Officers, and Works Safety Officers operating under CASA Part 139 in Australia and Norfolk Island.

---

## About this project

FlightOps Academy is a structured, self-paced handbook covering everything required to operate competently as a Flight Operations Officer at a certified aerodrome in Australia. It is built specifically around CASA Part 139 and the associated Manual of Standards, with additional content tailored to Norfolk Island Airport operations.

The handbook is designed for candidates who already hold an aviation background and are transitioning into airport-side operations roles.

## How to use

This repository is synced to GitBook. All content is written in Markdown and organised into numbered module folders under `/docs`.

To read the handbook, visit the GitBook site.
To contribute or update content, edit the relevant `.md` file and commit — GitBook updates automatically within 30 seconds.

## Regulatory basis

All content is written with reference to:
- CASR Part 139 — Aerodromes
- MOS Part 139 — Aerodrome Manual of Standards
- ICAO Annex 14 — Aerodromes
- ICAO Doc 9981 — PANS-Aerodromes
- Aeronautical Information Publication (AIP) Australia
- CASA Advisory Circulars
